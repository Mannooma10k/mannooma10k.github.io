Written By: Mannooma  |   Published: 12/10/25

Minecraft 26.10 Bedrock Changelog

The cutest drop in Minecraft history is here!

Ready or not, here comes the Tiny Takeover drop! New versions of baby mobs have stormed Minecraft, causing cuteness overload across biomes and builds. Listen out for heart-melting sounds, craft name tags for your pets, and keep babies forever young with the golden dandelion. The babies are officially on the loose, so get ready to collect, coo over, and create chaos with these tiny rascals!

Golden Dandelion

Introduced Golden Dandelion as a new type of flower that can be used to stop baby mobs from aging

  • Can be crafted using a Dandelion and Gold Nuggets
  • Can be used to craft Suspicious Stew
  • Can be crafted into Yellow Dye
  • Placed Golden Dandelion can be picked up by Endermen
  • Can be picked up by mobs who can pick up items
  • Can be used to distract Piglins
  • Displays green particles when used on a Baby Mob
    • Particles moving downwards indicate that aging is paused
    • Particles moving upwards indicate that age has been reset and is now resumed
  • Plays pause growth and reset growth sounds on use

Enabled the Golden Dandelion effect on the following Baby Mobs:

Baby Armadillo, Baby Axolotl, Baby Bee, Baby Camel, Baby Cat, Baby Chicken, Baby Cow, Baby Dolphin, Baby Donkey, Baby Fox, Ghastling, Baby Hoglin, Baby Goat, Baby Horse, Baby Llama, Baby Mooshroom, Baby Mule, Baby Nautilus, Baby Ocelot, Baby Panda, Baby Pig, Baby Polar Bear, Baby Rabbit, Baby Sheep, Baby Sniffer, Baby Strider, Baby Trader Llama, Baby Turtle, Baby Wolf, and Tadpole

Features and Bug Fixes

Accessibility

  • Fixed UI icons not showing at the lowest GUI scale setting (MCPE-234762)

Technical Updates Summary

This version includes extensive technical updates for:

  • AI Goals: Updated schemas for minecraft:behavior goals to be stricter when parsing. Entities with invalid data will now fail to load in versions 1.26.10 and newer.
  • API: Released multiple @minecraft/server APIs from beta to stable (v2.6.0 and v2.7.0-beta). Includes new Camera, World, and @minecraft/server-net functionality.
  • Blocks: Added support for multi-block structures, minecraft:multi_block trait, and minecraft:chest_obstruction component. Fixed rotation issues for blocks in item frames.
  • Character Creator: Added ban skin detection and improved appearance caching.
  • Client Entities: Added "held_item_scale" field to entity.json schema.
  • Commands: Fixed "/reload all" and "/setblock"/"/fill" commands issues.
  • Components: Replaced "simulate_waves" with "movement_type" in minecraft:buoyant. Added "can_auto_step_from_liquid" field.
  • Dedicated Server: Added HTTP configuration capabilities for @minecraft/server-net module.
  • Editor: Massive improvements including API updates, new timeline controls, jigsaw tool updates, and performance enhancements.
  • Entity Components: Added pause/reset growth functionality. Split minecraft:pushable into two new components.
  • Features: Fixed feature loading to support subdirectories.
  • Graphical: Colored block lighting improvements, texture streaming feature rollout, and various visual fixes.
  • JSON Schema: Updated canopy_decorator schema for block placement sequences.
  • Experimental: Voxel Shapes, Data-Driven UI (DDUI), and various other experimental features.

For complete technical documentation and detailed information, please visit the official Minecraft documentation.
