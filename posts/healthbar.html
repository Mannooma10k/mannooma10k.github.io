HEALTH BAR ADD-ON

Displays a customizable health bar above players and mobs.

About The Addon

It shows you a heart counter for monsters and mobs, and the nice thing about my addon is that it works with the rest of the mobs that in other Add-Ons

Add-ON Features

- Clean, Organized Name Tags.

- Different 3 Modes For Health Bar.

- Optimized Performance.

- Real Time Health Percentage Display.

How To Use

Default Mode

Default Mode is set to be 'Precentage Mode' and You will be able to change it throught the Menu.

Open HealthBar Modes

Make sure To enable Beta-api in your world to run the Add-On smoothly
then press on the Add-On icon
and you can change health bar Modes from there.

Installation

WINDOWS 10/11:

1. DOWNLOAD THE .MCPACK FILE.
2. DOUBLE-CLICK TO IMPORT.
3. ENABLE THE "BETA APIS" EXPERIMENT.
4. ACTIVATE BEHAVIOR PACK.

ANDROID / IOS (MCPE):

1. DOWNLOAD FILE.
2. OPEN WITH MINECRAFT.
3. GO TO WORLD SETTINGS > EXPERIMENTS.
4. YOU MUST ENABLE "BETA APIS"!
