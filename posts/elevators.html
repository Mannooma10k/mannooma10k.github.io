Elevators ADD-ON

KEEP IT EASIER!

About The Addon

Create seamless multi-floor elevators! Jump to go UP, Sneak to go DOWN. Now includes Public/Private modes, a specific "Trust" system for friends, and block camouflage. No redstone needed!

Add-ON Features

- Link unlimited floors regardless of the vertical distance.

- Restrict elevator access using public or private toggles.

- Disguise elevator blocks to match your surrounding builds.

- Manage network settings through custom interactive user menus.

How To Use

Craft Elevator Block

To make the Elevator Block,
You need 1x Wool, 1x ender_pearls

Add blocks

You can add More elevators by clicking on an elevator block then clicking on an existing one.

Main menu

After that place 2x Blocks and Right Click on two blocks Menu will appear Type your elevator Name
And toggle puplic button,Then Sneak + Right Click to Open Main Menu

You can change elevator status/delete the elevator & more through this menu

CamoFlage:
You can customize your elevators textures by selecting the block in your Inventory

Player Acess:
You can Give players Acess to control your elevator but cant delete it.

Players that have admin tag can control any elevator in the world/server, Even Delete it
