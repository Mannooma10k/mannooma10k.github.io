Installation

WINDOWS 10/11:

1. DOWNLOAD THE .MCPACK FILE.
2. DOUBLE-CLICK TO IMPORT.
3. ENABLE THE "BETA APIS" EXPERIMENT.
4. ACTIVATE BEHAVIOR PACK.

ANDROID / IOS (MCPE):

1. DOWNLOAD FILE.
2. OPEN WITH MINECRAFT.
3. GO TO WORLD SETTINGS > EXPERIMENTS.
4. YOU MUST ENABLE "BETA APIS"!
